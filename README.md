Source code for Illinois Manufactured Housing Communities Visualization Tool, based off of original Michigan version

*NOTICE*
Public website ONLY uses files in /deployment\
To edit/update the website, change files as needed in /deployment, then republish on Connect Cloud (access by logging in with this github account)\
(editing the duplicates outside of /deployment will NOT edit the website)

Update geospatial data by editing/replacing geojsons in /deployment/data, then run add_house_senate_districtsIL.R, add_homeRuleCommunties.R to update the csvs in deployment/data/  BEFORE republishing app.R on Connect Cloud
